# Rocky Linux Standardization Script

This repository contains a **Bash script** designed to apply **basic system configuration and security hardening** for Rocky Linux servers.  
The script helps automate standard settings and common security measures to ensure a stable and secure server environment.

---

## Features

- Basic system configurations (timezone, hostname, package updates, etc.)
- Security hardening (user and permission settings, SSH configuration, etc.)
- Removal or disabling of unnecessary services
- Log management and permission adjustments
- Other recommended baseline settings for Rocky Linux
